# メモ

- 検出処理だけ切り出してテスト
- その処理の入力は画像と動画
  - 画像と動画はAI生成
  - Google Flow候補
    - 画像はBanana
    - 動画はVeo(1回の会話で1本の動画)
- 開発過程をマークダウンで下書きして、LaTeXで清書

- 実装
  - 現場専用部分
    - グループ数表示コンポーネント
    - 非表示カメラ(ライブラリ使用)
    - 出し物情報と照らし合わせて、実際に出し物を利用するグループの数を計算
    - 外部送信

  - 研究専用部分
    - ファイルのアップロード
    - グループの数やバウンディングボックスなどの結果の描画
    - 結果のダウンロード

- 研究・現場共通部分
  - 手書きROI抽出コンポーネント(Canvas使用、入力された画像(TexImageSource型)を重ねて表示、返り値はCanvasの書き出したデータ)
  - グループの検出ロジック
    ※ ROIは既に処理されている前提
    - 🟥検出処理 (ここが肝)🟥
    - 取得メソッドで検出されたグループと人物を階層化されたオブジェクトで返す
      ```
      一覧のオブジェクト
        グループA
          人物A: bbox
          人物B: bbox
        グループB
          人物A: bbox
      ```
