# テスト用データに関するメモ

- 画像と動画の2種類
- 画像のシチュエーション
  - 基本: 普通に並んでいる状況（明るい場所 / 少し暗い場所）
  - 密集: 3〜4人のグループが固まって並んでいる状況
  - 他人同士: 知らない人同士がスマホを見ながら詰めて並んでいる状況
  - 様々な服装: クラスTシャツや、カチューシャ・帽子を被った状況
  - 画角の際: カメラの真下にいるグループと、カメラから一番遠くにいるグループ
 
- 動画のシチュエーション
  - 基本: 画像を動画に起こしただけのもの
  - 詰まり: 前のグループが詰まったときに、後ろの他人がグッと近づいて密着する状況
  - 広がり: グループの片方がスマホに夢中になり、グループ内の距離が一時的に2m以上離れてしまう状況
  - 現場のノイズ: 基本のシチュエーション + 手前の通路を人が行き交う
  - イレギュラーな行動: 列の途中で、後ろのグループが前のグループを追い越したり、横から1人が合流して3人グループになったり、列から1人が離脱（トイレなど）したりする状況
  - 文化祭特有のノイズ: 大きな看板やうちわ、風船を持ったグループが並ぶ。または、お揃いのクラスTシャツを着た集団が連続して並ぶ
