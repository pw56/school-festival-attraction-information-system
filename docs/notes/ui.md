# UIのメモ

## モバイル版
某テーマパークの本家アプリのように
1. サムネイル
2. 待ち時間
3. マップ
4. その他情報
のように表示する。
大まかな要素は次の通りである。
- サムネイル: 円形の画像、最も大きい要素
- 待ち時間: 最も大きいフォントサイズ
- マップ: アイコンとラベルのボタン、タップすると画面が暗転し、ポップアップで表示、画像の右上に閉じるボタンも
- その他情報: ラベルがH2で、その下にコンテンツ(テキスト)が表示される

## デスクトップ
基本的にモバイル版と同じ。
サムネイルと待ち時間とマップボタンを左側、その他の情報を右側にして、2カラムレイアウトにする。
