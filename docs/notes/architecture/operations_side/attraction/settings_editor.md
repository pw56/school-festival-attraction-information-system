# 設定エディター

- SPAのウェブアプリ
- 本部から配布された出し物登録QRコードで設定ページを開く
- GUIで直感的に出し物追加できる
- 出し物の情報を設定ファイルを直接編集することなく、GUIの編集ページで直感的に編集できる
- 出し物ごとの位置情報の登録も行い、手入力ではなくブラウザの現在地でも登録できる

- 会場(主に学校の校舎や体育館)の3Dマップの作成も行う(独自の簡易的な立体情報のみのモデル)
※ モデルは以下の情報のみを含む
  - 頂点の座標
  - 出し物の対応情報(結合キー)
※ 出し物の3D案内に使用される位置情報は団体ごとの自由度の観点から、マップのモデルではなく出し物ごと設定ファイルに含める
