# システム全体のダッシュボード

- ダッシュボードでは主に出し物の管理を行う
- 出し物の情報の登録用Zipのアップロードもここで行う
- システム全体の管理者はCloudflareアカウントでログイン

- 出し物の登録専用QRコードを発行
※ 文化祭でそこまで堅い認証は不要なのでURLにUUID使用
※ 出し物の情報の登録の一方通行で、取り消すには管理者側から
※ 他クラスへの妨害防止で、自分のクラスも含め取り消しはできない

- ダッシュボードで出し物ごとのシステム利用者用QRコードを発行
※ このページが、出し物ごとの運営も兼ねる
※ ダッシュボードから再発行できる、Cloudflareアカウントの使い回しを防止
※ 文化祭でそこまで堅い認証は不要なのでURLにUUID使用
