# School Festival Attraction Information System

## 💡 概要
文化祭で出し物の情報を掲載し、運営を効率化するためのシステム。

---

## 🚀 機能
- **PWA**: 来場者はシステムでホスティングされるウェブアプリを利用する。整理券の通知機能もある。
- **出し物の情報確認**: 出し物の出展団体が登録している場合は、運営時間やサムネイルなどの出し物の情報を確認することができる。
- **整理券**: 来場者は整理券を利用することで時間を捻出できる。スマホで取得できるデジタル版(TOTP版)と、案内所で取得できる紙版(トークン版)がある。
- **待ち時間確認**: 対応している出し物の待機列の待ち時間を確認できる。
- **3Dマップ**: 運営団体が登録している場合は、3Dマップを使用できる。出し物の位置も3次元的に確認できるようになる。
